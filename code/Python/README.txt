Setup Files:

ttyport.txt - Contains the name of the serial port for the blue tooth receiver.  For example /dev/ttyUSB0.
timeout.txt - Contains the measurement rate for all sensors.  The unit is wait time between measurements in increments of 15 seconds.  A value of 4 = 60 seconds wait, or 1 measurement per minute.
data_path.txt - Contains the path to the data files including the trailing \.  For example /home/user/GardenSense/Data/
